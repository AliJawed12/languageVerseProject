# LanguageVerse Project

LanguageVerse is a Node.js project that uses AI (OpenAI, Claude, Gemini) and MongoDB to generate language prompts and responses.

---

## Prerequisites

Before running the project, make sure you have:

1. **Node.js** (v18+ recommended) – [Download Node.js](https://nodejs.org/)  
2. **MongoDB** (local or cloud, e.g., MongoDB Atlas)  
3. **API Keys**:  
   - OpenAI → `OPENAI_API_KEY`  
   - Anthropic Claude → `ANTHROPIC_API_KEY`  
   - Google Gemini → `GEMINI_API_KEY`  

> ⚠️ **Important:** Do not commit `.env` files or any files containing secrets to GitHub.
